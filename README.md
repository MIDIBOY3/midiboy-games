- Godot 4.7 project files for the playable prototype

## GENESIS Engine Policy

`TSG_Godot` is treated as the source GENESIS game/engine. Project_ZAKO should inherit its top-down vertical shooter camera, altitude system, low-poly visual language, HUD grammar, and rendering assumptions before adding ZAKO-specific rules.

See:

- `docs/GENESIS_ENGINE.md`
- `docs/ZAKO_GAME_DESIGN.md`
